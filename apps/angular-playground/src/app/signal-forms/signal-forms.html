Wie funktioniert es?
  • form(signal({ name: '' }), (path) => { required(path.name) }) — erstellt das Signal-Formular
  • [field]="myForm.name" — bindet ein Feld an ein Input (kein ngModel nötig!)
  • myForm.name().invalid() — Zugriff auf den Validierungsstatus als Signal
  • submit(myForm, callback) — validiert und ruft callback nur bei gültigem Formular auf
