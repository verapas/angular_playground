Template-driven Forms

Formulare werden direkt im Template mit ngModel definiert.

FormsModule · ngModel · NgForm
@if (nameRef.invalid && nameRef.touched) { @if (nameRef.errors?.['required']) { Name ist erforderlich. } @if (nameRef.errors?.['minlength']) { Mindestens 2 Zeichen. } }
@if (emailRef.invalid && emailRef.touched) { @if (emailRef.errors?.['required']) { E-Mail ist erforderlich. } @if (emailRef.errors?.['email']) { Bitte eine gültige E-Mail eingeben. } }
@if (result()) {
✅ Formular abgesendet! 
{{ result() | json }}
}
Wie funktioniert es?
  • [(ngModel)] — Two-way Binding zwischen Input und Variable
  • #myForm="ngForm" — Template-Referenz auf das Formular-Objekt
  • #nameRef="ngModel" — Zugriff auf den Validierungsstatus des Feldes
  • Validierung via HTML-Attribute: required, minlength, email
